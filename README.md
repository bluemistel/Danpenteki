# Danpenteki - 会話フローエディタ

ホワイトボード型の会話フロー管理Webアプリケーション。登場人物の会話をノード（フィールド）としてキャンバス上に自由配置し、線で繋いで会話の流れを構成できます。ゲームシナリオや物語の会話フロー設計に最適です。

## 特徴

- **ホワイトボードキャンバス** — フィールド（会話ノード）を自由にドラッグ配置
- **会話ブロック編集** — 各フィールド内にキャラクターのセリフを複数追加
- **ノード接続** — フィールド間を線で繋いで会話の流れを定義
- **プレビューペイン** — 接続順に会話をリアルタイムプレビュー
- **キャラクター管理** — キャラクターの作成・アイコン設定・背景色カスタマイズ
- **CSV/TXTエクスポート** — 会話データを「話者,セリフ」形式で出力
- **アンドゥ・リドゥ** — Ctrl+Z / Ctrl+Y で操作の取り消し・やり直し
- **ブラウザローカル保存** — IndexedDBによる自動保存（サーバー不要）
- **アナログUIデザイン** — 紙・インク・テープをモチーフにした温かみのあるデザイン

## セットアップ

```bash
npm install
npm run dev
```

`http://localhost:3000` でアプリが起動します。

## 使い方

1. キャンバスをダブルクリックして新しいフィールドを追加
2. フィールド内の「ブロック追加」で会話ブロックを作成
3. キャラクターを選択してセリフを入力
4. フィールドのハンドル（上下左右の丸）をドラッグして別のフィールドに接続
5. フィールドをクリックすると右ペインにプレビュー表示
6. Alt+クリックでハンドルの接続を解除
